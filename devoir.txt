🎯 PROJET 2 : PANIER E-COMMERCE (45 min)
📋 CONSIGNES
Crée un panier avec ces produits :
const panier = [
  { nom: "Laptop", prix: 800, quantite: 1 },
  { nom: "Souris", prix: 25, quantite: 2 },
  { nom: "Clavier", prix: 60, quantite: 1 },
  { nom: "Écran", prix: 200, quantite: 2 },
  { nom: "USB", prix: 10, quantite: 5 },
]
Crée ces fonctions :

calculerTotal(panier) → retourne le prix total (prix × quantité pour chaque produit)

appliquerReduction(panier, pourcentage) → retourne le total après réduction

Exemple : appliquerReduction(panier, 10) → total - 10%


getProduitsChers(panier, seuil) → retourne les produits dont le prix dépasse le seuil

Exemple : getProduitsChers(panier, 100) → [Laptop, Écran]


getResume(panier) → retourne un tableau de strings formatés

Exemple : ["Laptop x1 = 800€", "Souris x2 = 50€", ...]


getProduitLePlusCher(panier) → retourne le produit avec le prix unitaire le plus élevé
